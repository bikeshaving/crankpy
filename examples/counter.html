Stateful component demonstrating event handling and state management.

JavaScript (React-style)

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <span>Count: {count}</span>
      <button onClick={() => setCount(count - 1)}>-</button>
      <button onClick={() => setCount(count + 1)}>+</button>
    </div>
  );
}

Python (Crank.py)

@component
def Counter(ctx):
    count = 0
    
    def increment():
        nonlocal count
        count += 1
        ctx.refresh()
    
    def decrement():
        nonlocal count
        count -= 1
        ctx.refresh()
    
    for _ in ctx:
        yield h.div[
            h.span[f'Count: {count}'],
            h.button(onClick=decrement)['-'],
            h.button(onClick=increment)['+']
        ]

Key differences:

  • State: Regular variables with nonlocal instead of useState
  • Updates: ctx.refresh() instead of setState
  • Rendering: Generator function with yield
  • JSX: h.div[children] syntax instead of <div>
  • Events: Direct function references instead of arrow functions
